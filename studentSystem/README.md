# 1.学生管理系统

### 1.1  需求分析

- 添加学生及成绩信息
- 将学生信息保存到文件中
- 修改和删除学生信息
- 查询学生信息
- 根据学生成绩进行排序
- 统计学生的总分

### 1.2  系统设计

- #####1.2.1  学生管理系统分为7大功能模块：如下

  - 学生信息模块
  - 查找学生信息模块
  - 删除学生信息模块
  - 修改学生信息模块
  - 学生成绩排名模块
  - 统计学生总人数模块
  - 显示全部学生信息模块
